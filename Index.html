How to Upload to GitHub:

  1. Create a new repository on GitHub named "calculator-project"
  2. Save this file as index.html
  3. Upload the file to your repository
  4. Enable GitHub Pages in your repository settings
  5. Your calculator will be live at: https://[your-username].github.io/calculator-project/
